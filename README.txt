Recipes Application

A Recipes app developed using using Node.js, Express.js, and Mongoose.
Along with Mongodb Atlas and postman for API requests.

Here CRUD opperations are implemented in local server.

A complete project created with MVC structure.
  1.db folder has connection.js for Mongodb connection with mongoose
  2.model folder with recipesModel.js for schema and validation
  3.controller folder has recipesController.js for request and response of the opperations with error handling 
  4.routes folder has recipesRoutes.js for the API calls with sample request and response from postman.
  5.index.js the root file with express imported and Server configurations.
